Langage utilisé: java
Interface graphique: javafx

Le projet est déjà pseudo compilé sinon vous pouvez le compiler
en exécutant la commande make dans le répertoire src

Par la suite exécutez le fichier "executer.bat"

Fonctionnement
Ce programme segmente un réseau ou un sous-réseaux en plusieurs sous-réseaux
Il affiche les adresses des sous-réseaux
l'adresse de broadcast ainsi que le nombre d'hôtes disponibles

Vous saisissez l'adresse ip
Ensuite vous choisissez s'il s'agit d'un réseau de base ou d'un sous réseau

Dans le cas où c'est un réseau, le masque de sous-réseau réseau est genéré automatiquement selon la classe de l'adresse
S'il s'agit d'un sous-réseaux, alors vous devrez completer avec les octets restants
